## 🎯 使用方式

### 基础使用
```bash
# 启动本地服务器
python3 -m http.server 8000

# 访问主网站
http://localhost:8000

# 查看使用指南
http://localhost:8000/management-guide.html

# 运行功能测试
http://localhost:8000/test_fixes.html
```

4. **数据备份**: 导出功能定期备份
