# AgricultureModeling 私人导航网站

## 项目简介

AgricultureModeling 是一个基于您的书签文件创建的私人导航网站，专门为农业建模研究人员设计。网站采用现代化的响应式设计，提供直观的分类浏览和强大的搜索功能。

## 功能特点

### 🌟 核心功能
- **分层导航**: 左侧树形导航，支持主分类和子分类
- **智能搜索**: 实时搜索书签标题和描述
- **双视图模式**: 网格视图和列表视图切换
- **响应式设计**: 完美适配桌面和移动设备
- **暗色主题**: 支持明暗主题切换
- **网站图标**: 自动获取网站favicon

### ⚙️ 管理功能
- **完整管理界面**: 可视化的书签管理系统
- **动态添加内容**: 支持添加主分类、子分类和书签
- **编辑和删除**: 右键菜单和管理按钮操作，精确删除指定项目
- **数据导入导出**: JSON格式的数据备份和恢复
- **本地数据存储**: 自动保存到浏览器本地存储
- **云端同步**: GitHub Gist 集成，静默后台同步
- **图标选择器**: 80+个精选图标供选择
- **拖拽重排**: 支持书签拖拽排序

## 使用方法

### 本地运行
1. 克隆或下载项目文件
2. 启动本地HTTP服务器:
   ```bash
   python3 -m http.server 8000
   ```
3. 在浏览器中访问: `http://localhost:8000`

### 管理功能使用
1. **进入管理模式**: 点击顶部工具栏的 ⚙️ 按钮
2. **添加分类**: 左侧面板点击"添加分类"按钮
3. **添加子分类**: 右键点击主分类，选择"添加子分类"
4. **添加书签**: 选择分类后，点击右侧"添加书签"按钮
5. **编辑删除**: 右键点击任意项目或使用管理按钮
6. **数据备份**: 使用导出功能定期备份数据

详细使用教程请查看: [📖 管理功能指南](./management-guide.html)

### 部署方案
- **静态托管**: 可部署到 GitHub Pages、Netlify、Vercel 等
- **服务器部署**: 可部署到任何支持静态文件的Web服务器
- **CDN加速**: 建议使用CDN加速字体和图标资源

## 自定义配置

### 添加新书签
1. 编辑 `data/bookmarks.json` 文件
2. 按照现有格式添加新的分类或链接
3. 刷新页面即可看到更新

### 主题定制
- 修改 `css/style.css` 中的CSS自定义属性
- 支持自定义颜色、字体、间距等

### 图标配置
- 使用 [Font Awesome](https://fontawesome.com/icons) 图标
- 在书签数据中修改 `icon` 字段
